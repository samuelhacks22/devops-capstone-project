# DevOps Capstone Project

This repository contains the **DevOps Capstone Project** for the **IBM DevOps and Software Engineering Professional Certificate**.  
The project demonstrates how to design, build, test, containerize, and deploy a RESTful microservice using modern DevOps practices and tools.

---

## Project Overview

The goal of this capstone project is to apply end-to-end DevOps principles, including:

- Agile development and sprint planning  
- Continuous Integration and Continuous Deployment (CI/CD)  
- Automated testing  
- Containerization with Docker  
- Deployment to Kubernetes  

The project follows real-world DevOps workflows from development to production.

---

## Key Features

- RESTful microservice built with Python and Flask  
- Automated unit and integration testing  
- CI/CD pipeline using GitHub Actions  
- Dockerized application  
- Kubernetes deployment with manifests  
- DevOps best practices and automation

| Technology | Purpose |
